# Page Override: Dishes

## Layout

- **Structure:** Search/Filter (sticky top) > Dish Grid

## DishCard

- Compact card with: Name (heading-3), Category badge, Calories, Rating stars
- Hover: shadow elevation increase

## Filter/Search

- Sticky below header on scroll
- Search input with search icon
- Category filter chips (horizontal scroll on mobile)
- Sort by: name, calories, rating
