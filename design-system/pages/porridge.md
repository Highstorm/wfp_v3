# Page Override: Porridge Calculator

> Overrides `design-system/MASTER.md` for the PorridgeCalculator.

## Layout

- **Structure:** Ingredient builder (left/top) > Nutrition preview (right/bottom)
- **Mobile:** Stack vertically (builder > preview)
- **Desktop:** Side-by-side layout

## Interaction

- Interactive ingredient selection with quantity sliders/steppers
- Live nutrition calculation updates
- Smooth value transitions on change (250ms)

## Style

- Playful but functional
- Emphasis on interactive elements
- Clear visual feedback when adding/removing ingredients
