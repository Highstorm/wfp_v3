    - For hasData: true days, uses actual eatenCalories and isStub: false

    CHART_COLORS:
    - Object with success, destructive, muted keys
    - Each key has light and dark HSL string values
    - Values match index.css CSS variables exactly

    - Constant array: ["Mo", "Di", "Mi", "Do", "Fr", "Sa", "So"]
